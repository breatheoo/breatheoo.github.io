# DESIGN.md

## 产品画像
"南宁张老师"——本地化的教育资源共享平台，面向南宁家长群体，提供免费学习资料与南宁学校信息。

## 气质与意象
- **意象锚点**：南宁本地一位热心的老教师，办公桌上摞着厚厚的试卷和练习册，耐心为家长解答学校信息。
- **气质关键词**：本地亲切、朴实可靠、实用不花哨、有求必应。

## 配色方案（清新暖黄风格）
| 色块 | 色值 | 用途 | 意象来源 |
|------|------|------|----------|
| 暖黄主色 | `#e8a838` | 按钮、装饰条、强调色 | 阳光温暖、教育活力 |
| 浅黄背景 | `#FFF5E6` | 卡片/标签底色 | 柔和奶油纸感 |
| 深灰文字 | `#4A4A4A` / `#333333` | 主标题、正文 | 舒适阅读、不刺眼 |
| 卡片底色 | `#FFFFFF` | 内容卡片底色 | 干净、清爽 |
| 浅灰文字 | `#888888` | 辅助文字 | 柔和低调 |
| 白色文字 | `#FFFFFF` | 按钮文字 | 清晰对比 |

## 字体排版
- **中文字体**：`"PingFang SC", "Microsoft YaHei", "Noto Sans SC"`（不加粗）
- **品牌名**：大号，黑色，无加粗，字间距紧凑
- **标签卡片**：圆角（12px），白底+浅阴影，暖黄装饰条，悬停上浮
- **主标题**：深灰（#333），中等字号
- **正文**：深灰（#4A4A4A），清晰柔和

## 页面结构（自上而下）
1. **顶部品牌区**：左上角图标 + "南宁张老师" 品牌标识
2. **主体内容区**（左右两栏）：
   - **左栏**：主标语 "免费学习资料分享"
   - **右栏**：三个服务标签卡片 "每日一练" "各校真题" "试卷解析"
3. **CTA 引导区**："点击下方按钮" / "添加张老师" 引导文案
4. **行动按钮**：暖黄色实心按钮，大圆角，白字
5. **二维码下拉区**：点击按钮后展开，展示微信二维码 + "长按识别二维码"

## 组件风格细节
- 标签卡片：圆角12px，白底，浅阴影，左侧暖黄竖条装饰，悬停上浮+阴影加深
- 主按钮：暖黄色实心，大圆角（50px），白字，悬停阴影加深
- 二维码：纯透明展示，无边框

## 动效与交互
- 页面加载后主体柔和上浮（fadeIn + translateY）
- 三个标签卡片依次淡入
- 标签卡片悬停：上浮 + 阴影加深
- 按钮悬停：阴影加深 + 轻微放大
- 二维码：点击按钮后下拉展开（弹性动画），再次点击收起

## 设计禁忌
- 不使用粗边框（放弃鱼圈粗边框风格）
- 不使用深棕色文字，改用深灰
- 保持干净清爽，避免花哨
- 背景纯白，不用暖色底纹
